Tao Controller day du (Resource controller)

php artisan make:controller PostController --resource dk Route : Route::resource('posts', 'PostController');

Tao Controller day du (Resource controller) dong thoi tao MODEL

php artisan make:controller PhotoController --resource --model=Photo

Partial resource routes

Voi tu khoa except va only se lay cac route can thoi Route::resource('posts', 'PostController')->only([ 'index', 'show' ]); -> chi lay cac route can thiet (index va2 show) // Đăng ký tất cả các resource route trong PhotoController ngoại trừ destroy Route::resource('photos', 'PhotoController')->except([ 'destroy' ]); -> loai bo deytroy
